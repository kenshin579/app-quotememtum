# 개인정보처리방침 (Privacy Policy)

**InspireMe Chrome Extension**

최종 수정일: 2026년 4월 6일

## 1. 개요

InspireMe("본 확장 프로그램")는 Chrome 새 탭에서 영감을 주는 명언과 배경 이미지를 표시하는 확장 프로그램입니다. 본 개인정보처리방침은 본 확장 프로그램의 데이터 수집 및 사용에 관한 정책을 설명합니다.

## 2. 수집하는 데이터

본 확장 프로그램은 **개인 정보를 수집하지 않습니다.**

## 3. 로컬 저장 데이터

본 확장 프로그램은 Chrome Storage API를 사용하여 다음 설정을 **사용자의 기기에만** 로컬로 저장합니다:

- 시계 형식 (12시간/24시간)
- 명언 모드 (오늘의 명언/랜덤)
- 명언 갱신 주기
- 명언 폰트 및 폰트 크기
- 언어 설정 (한국어/English)
- 다크 모드 설정

이 데이터는 외부 서버로 전송되지 않으며, 사용자의 브라우저에만 저장됩니다.

## 4. 외부 API 통신

본 확장 프로그램은 기능 제공을 위해 다음 외부 서비스와 통신합니다:

- **InspireMe API** (inspire-me.advenoh.pe.kr): 명언 데이터를 가져오기 위해 사용됩니다.
- **Unsplash API** (api.unsplash.com): 배경 이미지를 가져오기 위해 사용됩니다.

이 통신 과정에서 개인 식별 정보는 전송되지 않습니다.

## 5. 제3자 데이터 공유

본 확장 프로그램은 사용자 데이터를 제3자에게 판매, 전송 또는 공유하지 않습니다.

## 6. 권한 사용 목적

| 권한 | 목적 |
|------|------|
| `storage` | 사용자 설정을 로컬에 저장 |
| `host_permissions` (inspire-me.advenoh.pe.kr) | 명언 데이터 API 호출 |
| `host_permissions` (api.unsplash.com) | 배경 이미지 API 호출 |
